# ToPu_Resample

**ToPu_Resample** は、Blender のメッシュ編集で、選択した辺まわりの割り数や形の流れを手早く整えるためのアドオンです。

トポロジーを大きく作り直さず、必要な部分だけ **増やす・減らす・整える・なめらかにする** ことを目的にしています。

このアドオンは **Anton Vashkevich** 氏による **AV: ResampleMesh v1.1.9** を元にした改変版で、**GPL-3.0-or-later** として公開しています。

---

## できること

- メッシュの割り数をあとから増やす
- 多すぎる割りを減らして整理する
- 頂点の間隔をそろえる
- 丸みのある部分を自然に整える
- ガタついたラインや輪郭をスムーズにする

---

## こんな時に便利です

| やりたいこと | 使う機能 |
| --- | --- |
| 丸みが足りない部分の割りを増やしたい | リサンプル（Resample） |
| 割りが多すぎる部分を減らしたい | リサンプル（Resample） |
| 頂点の間隔をきれいにそろえたい | リサンプル（Resample）+ 均等配置（Even Spacing） |
| 形を大きく変えずに頂点数だけ調整したい | リサンプル（Resample）+ 形状維持（Shape Preserve） |
| ガタついた線や輪郭をなめらかにしたい | スムーズ（Smooth） |
| スムーズにしたいが、縮みすぎるのを抑えたい | スムーズ（Smooth）のシャープ（Sharp） |

---

## 対応環境

- Blender `4.1` 以降を想定
- メッシュオブジェクトの編集モードで使用します

---

## インストール方法

1. このリポジトリから ZIP ファイルをダウンロードします。
2. Blender を開きます。
3. `編集 > プリファレンス > アドオン` を開きます。
4. `インストール...` または `ディスクからインストール...` を押します。
5. ダウンロードした ZIP ファイルを選択します。
6. **ToPu_Resample** を有効化します。

古い ToPu_Resample や元になったリサンプル系アドオンが入っている場合は、先に無効化または削除してからインストールすることをおすすめします。

---

## 基本の使い方

1. メッシュオブジェクトを選択します。
2. 編集モードに入ります。
3. 調整したい辺を選択します。
4. 右クリックメニューから目的の機能を実行します。

| メニュー | 内容 |
| --- | --- |
| `ToPu Tools > Resample` | 割り数を増やす・減らす・頂点間隔を整える |
| `ToPu Tools > Smooth` | 選択部分のガタつきをなめらかにする |

実行中は 3D ビュー上にプレビューが表示されます。結果を見ながら調整し、問題なければ `Enter` または `Space` で確定します。やめたい場合は `Esc` または右クリックでキャンセルできます。

## リサンプル（Resample）

リサンプル（Resample）は、選択した辺の流れに沿って、頂点数や頂点の並びを調整する機能です。

「今の形は残したいけれど、割り数だけ変えたい」「頂点の間隔を整えたい」という場面で使います。

### よく使う操作

| 操作 | 内容 |
| --- | --- |
| `Shift + ホイール` | 頂点数を増減します |
| `Shift + Q` / `Shift + E` | キーボードで頂点数を増減します |
| `C` | 補間方式（Interpolation Mode）を切り替えます |
| `T` | 均等配置（Even Spacing）を切り替えます |
| `S` | サブディビジョンモード（Subdivision Mode）を切り替えます |
| `Ctrl + ホイール` | 形状維持（Shape Preserve）または張力（Tension）を調整します |
| `Ctrl + Q` / `Ctrl + E` | キーボードで形状維持（Shape Preserve）または張力（Tension）を調整します |
| `R` | 実行中の設定をリセットします |
| `Enter` / `Space` | 確定します |
| `Esc` / 右クリック | キャンセルします |

### 補間方式（Interpolation Mode）の選び方

| 補間方式 | 使いどころ |
| --- | --- |
| 曲率補正（Curvature Correction） | 迷ったらまずこれ。丸みや流れを自然に保ちながら調整できます。 |
| スムーズ補間（Smooth Interpolation） | 柔らかい曲線のように、なめらかにつなげたい時に使います。 |
| 形状維持（Keep Shape） | 元の形をなるべく残したまま、頂点数だけ変えたい時に使います。 |
| 張力制御（Tension Control） | カーブの膨らみや締まり具合を細かく調整したい時に使います。 |

### 形状維持（Shape Preserve）

曲率補正（Curvature Correction）使用時は、形状維持（Shape Preserve）で角の残り具合を調整できます。

| 値の目安 | 挙動 |
| --- | --- |
| `0.00` | なめらかで自然な補正になります |
| `1.00` 前後 | 元の折れ線感や角を残しやすくなります |
| `1.00` 以上 | 角ばった形状をより強く残します |

角を残したい場合は、曲率補正（Curvature Correction）のまま、`Ctrl + ホイール` で形状維持（Shape Preserve）を上げて調整してください。

## スムーズ（Smooth）

スムーズ（Smooth）は、頂点数を変えずに、選択した部分のガタつきやラインの乱れをなめらかに整える機能です。

リサンプル（Resample）が「割り数を調整する機能」なのに対して、スムーズ（Smooth）は「形の流れを整える機能」です。

### よく使う操作

| 操作 | 内容 |
| --- | --- |
| `Shift + ホイール` | 反復回数（Iterations）を増減します |
| `Shift + Q` / `Shift + E` | キーボードで反復回数（Iterations）を増減します |
| `Ctrl + ホイール` | 強度（Factor）を調整します |
| `Ctrl + Q` / `Ctrl + E` | キーボードで強度（Factor）を調整します |
| `S` | スムーズタイプ（Smooth Type）を切り替えます |
| `W` | ウェイト（Weights）を切り替えます |
| `B` | 境界固定（Lock Boundary）を切り替えます |
| `R` | 実行中の設定をリセットします |
| `Enter` / `Space` | 確定します |
| `Esc` / 右クリック | キャンセルします |

### スムーズタイプ（Smooth Type）

| タイプ | 内容 |
| --- | --- |
| スムーズ（Smooth） | 標準的なスムーズです。ガタつきを素直にならします。 |
| シャープ（Sharp） | 形が縮みすぎるのを抑えながら整えます。輪郭を残したい時に便利です。 |

## アドオン設定

`編集 > プリファレンス > アドオン > ToPu_Resample` から設定できます。

| 設定 | 内容 |
| --- | --- |
| 言語（Language） | アドオン内の表示言語を変更できます。自動（Auto）/ 日本語（Japanese）/ 英語（English）から選べます。 |
| 実行後に法線をリセット（Reset Normals After Run） | 実行後に法線をリセットします。カスタム法線を使うモデルではオフ推奨です。 |
| 頂点数プレビューを表示（Show Vertex Count Preview） | 3D ビュー上に頂点数のプレビューを表示します。 |
| 操作ヒントを表示（Show Operation Hints） | 実行中のショートカットヒントを表示します。 |
| HUD 表示設定 | ヒント位置、表示位置、点サイズ、文字サイズ、文字色などを調整できます。 |

## うまく動かない時

### メニューが見つからない

メッシュオブジェクトの編集モードで右クリックしてください。`ToPu Tools` メニュー内に `Resample` と `Smooth` が表示されます。

### 実行できない

以下を確認してください。

- メッシュオブジェクトの編集モードになっているか
- 辺を選択しているか
- 選択範囲が複雑になりすぎていないか

うまく認識されない場合は、調整したい部分だけをシンプルに選択し直してから実行してください。

### カスタム法線が変わる

実行後に法線をリセット（Reset Normals After Run）がオンの場合、法線がリセットされます。カスタム法線を使っているモデルでは、この設定をオフにしてください。

### 古いアドオンと競合する

同名のアドオンや派生元のアドオンが有効なままだと、メニューやオペレーターが競合する場合があります。古い版を無効化または削除してから再インストールしてください。
