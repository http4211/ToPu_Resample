# ToPu_Resample

このアドオンは **Anton Vashkevich** 氏による **AV: ResampleMesh v1.1.9** を元にした改変版でリサンプル機能に特化したアドオンとなっています。**GPL-3.0-or-later** として公開しています。

---

## 概要

ToPu_Resample は、Blender のメッシュ編集時に、選択した辺を適切な形で頂点数を調整するアドオンです。

主に以下のような用途を想定しています。

- 形状の丸みなどが足りないときの、メッシュの割を増減したい時
- 特定の一部分だけ、メッシュの割を増減したいとき

---

## 対応環境

- Blender: `4.1` 以降を想定
- 動作対象: メッシュオブジェクトの編集モード

---

## インストール方法

1. このリポジトリから ZIP ファイルをダウンロードします。
2. Blender を開きます。
3. `編集 > プリファレンス > アドオン` を開きます。
4. `インストール...` または `ディスクからインストール...` を選択します。
5. ダウンロードした ZIP ファイルを選択します。
6. **ToPu_Resample** を有効化します。

### 既存版から更新する場合

過去の `ToPu_Resample` や派生元のリサンプル系アドオンを入れている場合は、競合を避けるため、先に古いアドオンを無効化または削除してからインストールすることをおすすめします。

---

## 基本的な使い方

1. メッシュオブジェクトを選択します。
2. 編集モードに入ります。
3. リサンプルしたいエッジ列、またはエッジループを選択します。
4. 右クリックメニューから `ToPu Tools > Resample` を実行します。
5. モーダル操作中にスクロールやショートカットで頂点数、補間方式、形状維持などを調整します。
6. 結果が問題なければ `Space` または `Enter` で確定します。
7. キャンセルしたい場合は `Esc` または右クリックを押します。

---

## 操作方法

| 操作 | 内容 |
| --- | --- |
| `Shift + Wheel` | 頂点数を増減します。 |
| `Shift + Q` / `Shift + E` | 頂点数をキーボードで増減します。 |
| `C` | 補間方式を切り替えます。 |
| `S` | 通常の頂点数調整とサブディビジョンモード（Subdivision Mode）を切り替えます。 |
| `T` | 均等配置（Even Spacing）のオン/オフを切り替えます。 |
| `R` | モーダル中の設定をリセットします。 |
| `Ctrl + Wheel` | 曲率補正（Curvature Correction）時は形状維持（Shape Preserve）、張力制御（Tension Control）時は張力（Tension）を調整します。 |
| `Ctrl + Q` / `Ctrl + E` | 形状維持（Shape Preserve）または張力（Tension）をキーボードで調整します。 |
| `Space` / `Enter` | 現在のプレビュー結果で確定します。 |
| `Esc` / 右クリック | 操作をキャンセルします。 |

## 補間方式（Interpolation Modes）

### 曲率補正（Curvature Correction）

デフォルトの補間方式です。

エッジ列やループの曲率を考慮しながら、できるだけ自然な流れで頂点を再配置します。曲線的な形状、閉じたループ、丸みのあるパーツの調整に向いています。

この方式では、形状維持（Shape Preserve）を使って角や折れ線感の残り具合を調整できます。

### スムーズ補間（Smooth Interpolation）

全体をなめらかにつなぐことを重視した補間方式です。

滑らかなカーブ状のエッジ列や、柔らかい面の流れを整えたい場合に向いています。角を強く残すよりも、自然なカーブとして整えたい場合に使いやすい方式です。

### 形状維持（Keep Shape）

元のエッジ列の流れや形状を比較的維持しながら補間する方式です。

大きく形を変えずに頂点数だけ調整したい場合や、元のトポロジーの印象を残したい場合に向いています。

### 張力制御（Tension Control）

張力（Tension）パラメーターで補間の硬さを調整できる方式です。

モーダル操作中に `Ctrl + Wheel` または `Ctrl + Q/E` を使うことで、補間の張り具合を調整できます。カーブの膨らみや締まり具合を細かく調整したい場合に便利です。

---

## 形状維持（Shape Preserve）

形状維持（Shape Preserve）は、曲率補正（Curvature Correction）使用時に有効なパラメーターです。

値を上げるほど、角や折れ線的な形状を維持しやすくなります。

| 値 | 挙動の目安 |
| ---: | --- |
| `0.00` | 従来の曲率補正に近い、なめらかな挙動になります。 |
| `1.00` | 元の折れ線形状へ寄り、角を維持しやすくなります。 |
| `1.00` 以上 | 角や硬い形状の保持をさらに強めます。 |

形状維持（Shape Preserve）は、モーダル操作中に `Ctrl + Wheel` または `Ctrl + Q/E` で調整できます。

---

## 均等配置（Even Spacing）

均等配置（Even Spacing）をオンにすると、選択したエッジ列やループ上の頂点分布をできるだけ均等に整えます。

頂点間隔をそろえたい場合や、リトポロジー作業で見た目の分布を整えたい場合に便利です。

`T` キーでオン/オフを切り替えできます。

---

## サブディビジョンモード（Subdivision Mode）

`S` キーで、通常の頂点数調整とサブディビジョンモード（Subdivision Mode）を切り替えできます。

通常モードでは、選択エッジ列やループの頂点数を `+1` / `-1` のように増減します。

サブディビジョンモード（Subdivision Mode）では、元のエッジ数を基準に、段階的に分割または削減するような感覚で調整できます。

---

## 選択状態ごとの挙動

### 開いたエッジ列（Open Edge Chain）

開いたエッジ列を選択して実行すると、端点を維持しながらエッジ列全体の頂点数を調整します。

ライン状のパーツ、境界エッジ、途中で閉じていないエッジ列の調整に向いています。

### 閉じたループ（Closed Loop）

閉じたループを選択して実行すると、ループとしての接続を保ちながら頂点数を調整します。

円柱の断面、口・目・穴まわりのループ、輪郭状のトポロジーの調整に向いています。

### 混合選択（Mixed Selection）

開いたエッジ列と閉じたループを同時に選択した状態にも対応しています。

ただし、分岐した選択や複雑すぎる選択状態では、連続エッジ列として判定できず処理を中断する場合があります。その場合は、分岐部分を外して、シンプルなエッジ列またはループにしてから実行してください。

## アドオン設定

`編集 > プリファレンス > アドオン > ToPu_Resample` から設定できます。

| 設定 | 内容 |
| --- | --- |
| 言語（Language） | アドオン内の表示言語を設定します。 |
| 実行後に法線をリセット（Reset Normals After Run） | 実行後に法線をリセットします。カスタム法線を使うモデルでは注意してください。 |
| 頂点数プレビューを表示（Show Vertex Count Preview） | 3Dビュー上に頂点数プレビューを表示します。 |
| 操作ヒントを表示（Show Operation Hints） | モーダル操作中のヒントHUDを表示します。 |
| ヒント位置Y（Hint Position Y） | ヒントHUDの縦位置を調整します。 |
| ヒントを上側に表示（Show Hints at Top） | ヒントHUDを画面上側に表示します。 |
| プレビュー点サイズ（Preview Point Size） | プレビュー点のサイズを調整します。 |
| 頂点数テキストサイズ（Vertex Count Text Size） | 頂点数表示の文字サイズを調整します。 |
| 頂点数テキストカラー（Vertex Count Text Color） | 頂点数表示の色を調整します。 |

### 言語（Language）

言語（Language）は以下から選択できます。

| 値 | 内容 |
| --- | --- |
| 自動（Auto） | Blender 本体の UI 言語に追従します。 |
| 日本語（Japanese） | ToPu_Resample の表示を日本語に固定します。 |
| 英語（English） | ToPu_Resample の表示を英語に固定します。 |

## トラブルシューティング

### メニューが見つからない

メッシュオブジェクトの編集モードで右クリックメニューを開いてください。

アドオンは `ToPu Tools > Resample` に追加されます。他の ToPu 系アドオンが入っている場合は、既存の `ToPu Tools` メニュー内に統合されることがあります。

### 実行できない / エラーが出る

以下を確認してください。

- メッシュオブジェクトの編集モードになっているか
- エッジが選択されているか
- 選択が分岐していないか
- 開いたエッジ列（Open Edge Chain）または閉じたループ（Closed Loop）として認識できる形になっているか

### カスタム法線が変わる

実行後に法線をリセット（Reset Normals After Run）がオンになっている場合、実行後に法線がリセットされます。

カスタム法線を使っているモデルでは、この設定をオフにしてください。

### 古いバージョンと競合する

同名のアドオンや派生元のアドオンが有効なままだと、メニューやオペレーターが競合する可能性があります。

古い版を無効化または削除してから再インストールしてください。
