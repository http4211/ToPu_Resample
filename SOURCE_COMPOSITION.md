# ソースコード構成の概算

ToPu_Resample は、**AV: ResampleMesh v1.1.9** を参考に作成した Blender アドオンです。

このファイルでは、公開時の透明性のため、参考元アドオンと現在の ToPu_Resample の Python コードを行単位で比較した概算を記載しています。

---

## 重要な注意

この数値は、著作権上の作者比率、権利範囲、ライセンス範囲を法的に確定するものではありません。

完全一致している行だけを機械的に数えているため、以下のような要素は数値に正確には反映されません。

- 参考元コードを変更した行
- ファイル構成や関数名を変更したコード
- UI 文言の日本語化・英語化
- 挙動を参考にしつつ ToPu_Resample 向けに作り直した処理
- スムーズ（Smooth）機能の追加・調整部分
- リサンプル（Resample）の操作体系や HUD 表示の調整

そのため、このファイルは「どの程度の完全一致行が残っているか」を示すための参考資料です。

---

## 計測方法

- 比較元: `AV: ResampleMesh v1.1.9`
- 比較先: 現在の `ToPu_Resample`
- 対象: Python ソースファイルのみ（`*.py`）
- 除外: 空行、コメントのみの行、docstring のみの行
- 一致判定: 空白差分をならした行単位の完全一致
- 重複行: 同じ行が複数ある場合は、出現回数も考慮して概算
- 対象外: `README.md`、`NOTICE.txt`、`LICENSE`、画像、ZIP 構成などの非 Python ファイル

## 実務上の見方

`modules/func.py`、`modules/splines.py`、`utils/draw.py` などの補助的な処理には、参考元と一致する行が多く残っています。

一方、`modules/topu_modal.py` では、ToPu_Resample 向けにリサンプル（Resample）の操作、HUD、補間方式、形状維持、混合選択時の処理などを大きく整理・拡張しています。

また、`modules/topu_smooth.py` では、スムーズ（Smooth）機能を ToPu_Resample の右クリックメニュー、モーダル操作、HUD 表示に合わせて追加・調整しています。
